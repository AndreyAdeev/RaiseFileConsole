# RaiseFileConsole
To save files by URL


Прога для сохранения файла по URL адресу (версия 1.0)
- введите в консоли (путь - папка с файлом RaiseFile.class) команду: java RaiseFile;
- нажмите Enter;
- введите URL файла;
- нажмите "Enter";
- введите название нового файла;
- нажмите "Enter";

По умолчанию файл сохранится в папку с файлом RaiseFile.class.
Для сохранения в другую папку введите имя файла с указанием диска и папки. 
Пример - D:\имя папки\имя файла.расширение
